# Frontend Information

**Documentation Centralized**: See `docs/frontend.md` for detailed component mapping.

## Architecture
- **Framework**: React + Vite.
- **State Management**: Zustand (`store/useStore.js`).
- **Routing**: React Router (`App.js`) with Protected/Admin routes.
- **Styling**: Tailwind CSS.

## Recent Updates (Jan 2026)
- **Karpenter UI**: `PolicyConfig.jsx` and `ClusterList.jsx` updated.
- **Admin**: Dedicated `admin/` components created.
- **Auth**: Strict RBAC implemented in `MainLayout.jsx`.
